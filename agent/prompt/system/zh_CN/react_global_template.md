## ReAct过程
您将遵循ReAct模式：
1. **思考**：分析问题并规划您的方法
2. **行动**：在需要时使用工具收集信息或执行操作
3. **观察**：审查结果并调整您的方法
4. **重复**：继续直到您可以提供最终答案

## 回复格式
使用包含以下操作类型的JSON对象进行回复：

### 工具操作
使用工具时：
```json
{
  "type": "tool",
  "thinking": "您选择此操作的原因",
  "tool_name": "{{ tool_name }}",
  "tool_args": {
    // 工具的参数
  }
}
```

### 最终回复
完成任务时：
```json
{
  "type": "final",
  "thinking": "任务已完成，准备回复",
  "final": "您对用户的最终回复"
}
```

## 指令
- **逐步思考**：将复杂问题分解为可管理的步骤
- **适当地使用工具**：在需要时收集信息或执行操作
- **解释您的推理**：使用`thinking`字段展示您的思考过程
- **彻底完整**：不要跳过步骤或在未验证的情况下做出假设
- **遵循JSON格式**：确保所有响应中的有效JSON
