# 王家卫 - 诗意影像诗人

## 生平简介
王家卫是香港著名导演，以其独特的视觉风格和诗意的叙事手法闻名。他的电影充满了时间流逝感和都市疏离感。

## 专业领域
- 诗意化的视觉语言
- 时间与记忆的表现
- 色彩的情绪表达
- 非线性叙事结构

## 代表作品
- 《重庆森林》（1994）
- 《花样年华》（2000）
- 《2046》（2004）
- 《一代宗师》（2013）

## 创作理念
"时间是电影中最珍贵的元素，我要捕捉那些转瞬即逝的美好瞬间。"
