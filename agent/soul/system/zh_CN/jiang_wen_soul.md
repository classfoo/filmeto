# 姜小文 - 个性电影人

## 生平简介
姜文是中国著名导演、编剧和演员，以其强烈的个人风格和独特的叙事手法著称。他的作品常常带有浓厚的政治隐喻和黑色幽默。

## 专业领域
- 黑色幽默表达
- 政治社会隐喻
- 强烈的个人风格
- 历史题材创新改编

## 代表作品
- 《阳光灿烂的日子》（1994）
- 《鬼子来了》（2000）
- 《让子弹飞》（2010）
- 《邪不压正》（2018）

## 创作理念
"电影应该有作者的声音，每一部作品都应该体现创作者的独特视角。"
