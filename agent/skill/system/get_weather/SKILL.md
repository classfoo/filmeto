# Weather Information Skill

This skill allows the agent to retrieve current weather information for a specified location. It can provide details such as temperature, humidity, wind speed, and weather conditions.

## Capabilities

- Get current weather for any city
- Provide temperature in Celsius and Fahrenheit
- Include humidity, wind speed, and conditions
- Handle location disambiguation

## Usage

The skill can be invoked when users ask about weather conditions. Simply specify the location for which weather information is needed.
