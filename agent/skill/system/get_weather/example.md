# Weather Skill Examples

## Basic Usage

**Input:** "What's the weather like in Beijing?"

**Output:** 
```
The current weather in Beijing is partly cloudy with a temperature of 22°C (71.6°F). 
Humidity is at 65% and there is a wind speed of 12 km/h.
```

## Detailed Request

**Input:** "How hot is it in Dubai today?"

**Output:**
```
Currently in Dubai, it's 38°C (100.4°F) with sunny conditions. 
Humidity is at 45% and there is a light breeze of 8 km/h.
```

## Forecast Request

**Input:** "Will it rain tomorrow in London?"

**Output:**
```
According to the forecast, there's a 30% chance of rain in London tomorrow. 
Expected temperature range is 15-18°C (59-64°F) with partly cloudy conditions.
```
